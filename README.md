# AI_assistant_for_browers
浏览器插件,主要功能是登录爱创账户,并能将当前网址的文章导入到爱创收藏中

## Tampermonkey脚本
### 安装Tampermonkey浏览器插件
[Tampermonkey官网][1]选择自己浏览器对应插件,并安装
>  chrome浏览器需自备梯子,不会翻墙的可选择[下载插件离线安装][2],[离线安装方法][3]

### 安装爱创收藏脚本
[脚本下载地址][4]
### 保存商品详情页内容
进入淘宝/天猫/京东产品详情页面,Tampermonkey脚本会自动检测运行,点击页面购买按钮旁的**保存到爱创**按钮
